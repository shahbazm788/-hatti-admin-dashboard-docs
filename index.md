#  üöÄ  Hatti Modern React 19 eCommerce Admin Dashboard ‚Äì Introduction & Overview

![Home Dashboard](/images/HomeDark.png)
**Hatti** is a clean, modular, and high-performance admin dashboard template designed for modern eCommerce management. Built with the latest **React 19**, **Vite**, and **Tailwind CSS v4**, it provides a solid and flexible foundation for developers to build everything from small store backends to complex business management tools.

Instead of a cluttered and heavy framework, Hatti focuses on a **developer-first architecture**. It offers a curated set of UI components, interactive analytics, and pre-built applications like Kanban and Chat, allowing you to kickstart your frontend development without spending weeks on setup.

Hatti is built to be **extensible**. With its clear folder structure and state management via **Zustand**, you can easily integrate your own APIs and scale the dashboard as your business requirements grow.

## üöÄ Key Features

- **Built with React 19 & Vite** Experience the next generation of React with ultra-fast build times and modern rendering.

- **Tailwind CSS v4 Styling** Fully customizable using the latest CSS-variable driven engine for effortless branding and theming.

- **Interactive Data Widgets** Engaging and responsive charts powered by **Recharts** to visualize your business KPIs and sales data.

- **Ready-to-Use UI Kit** A comprehensive library of modular, atomic components (Tables, Forms, Modals) specifically designed for eCommerce.

- **Productivity-Ready Apps** Includes pre-designed modules for Kanban boards, real-time Chat interfaces, Calendars, and Order management views.

- **Scalable Codebase** Clean, documented, and type-safe architecture using **TypeScript** to ensure long-term maintainability.
